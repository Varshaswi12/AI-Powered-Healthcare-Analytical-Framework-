# 🏥 Healthcare Analytics SaaS

An AI-powered Healthcare Analytics Platform that allows users to query healthcare data using natural language and generate real-time insights, analytics, and reports.

---

## 🚀 Features

- 🔎 Natural Language to SQL (LLM-based)
- 📊 Interactive Dashboard with Charts (Chart.js)
- 📈 KPI Metrics (Total Patients, Average Age, Top Condition)
- 🧠 Dataset Insights & Exploratory Analysis
- 📥 Download Query Results as CSV
- 📄 Generate PDF Healthcare Reports
- 🤖 AI-powered Query Understanding (Groq LLM)

---

## 🏗️ Tech Stack

- **Backend:** Flask (Python)
- **Database:** SQLite
- **Frontend:** HTML, CSS, Chart.js
- **AI Integration:** Groq LLM (LLaMA 3.3 70B)
- **Data Processing:** Pandas
- **ML Model:** Scikit-learn
- **PDF Generation:** ReportLab
